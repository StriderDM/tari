# Tari p2p examples

Examples for using the `tari_p2p` crate.

To run:

```bash
cargo run --example $name -- [args]
```

---

## Examples

### [`peer_editor.rs`](peer_editor.rs)

Edit a peer list file.

*Dependencies: ncurses*

```bash
cargo run --example peer_editor -- --file=./path/to/peers/file.db
```
